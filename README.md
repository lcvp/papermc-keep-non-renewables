# PaperMC Keep Non-Renewables

> [!NOTE]
> This is for Minecraft **1.21.11**.

PaperMC config to prevent non-renewable resources from despawning.

## Why?

Vanilla Minecraft has items that [cannot be farmed indefinitely][minecraft-wiki-non-renewable-resource], so obviously for long term worlds you'd want to lose as few of them as possible.

[minecraft-wiki-non-renewable-resource]: https://minecraft.wiki/w/Non-renewable_resource
